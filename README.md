# 🌿 MoodCircle App – UI/UX Design Project

## 📌 Overview
MoodCircle is a mental wellness mobile application designed to help users track their mood, understand emotions, and improve mental well-being through journaling and daily reflection.

---

## 🎯 Features Covered
- User Research to identify user needs
- Creation of User Personas
- User Flow Design
- Low-Fidelity Wireframes (Figma)
- Design Thinking Process

---

## 👥 User Research
Identified that users need a simple way to track mood, reduce stress, and maintain emotional balance using a clean and easy interface.

---

## 🧑‍💼 User Personas
- Student users dealing with academic stress
- Young professionals facing work pressure

---

## 🔄 User Flow
Login → Dashboard → Mood Selection → Energy Level → Journal Entry → Goals → History
